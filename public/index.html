How to Use mark2

mark2 creates mind maps organized in radial hierarchy, with different colors for each level:

Hierarchical Structure:

  • Center: Main title (#) - core of the map
  • First level: Sections (##) in blue - connected to center
  • Second level: Subsections (###) in purple - children of sections
  • Third level: Items (-) in green - children of subsections

Support Text:

Text without markup serves for:

  • ‚Ä¢ Documentation and detailed explanations
  • ‚Ä¢ Context and complementary information
  • ‚Ä¢ Reference material and support

This text does not appear as nodes in the map, but helps understand the content.

Markdown Elements:

  • Use # for the central topic (appears at the center of the map)
  • Use ## for main sections (blue nodes around the center)
  • Use ### for subsections (purple nodes, children of sections)
  • Use - for specific items (green nodes, children of subsections)
  • Use numbers for ordered lists
  • Plain text serves as documentation and context (does not become visual nodes)

Navigation:

  • Use zoom controls to navigate the map
  • Drag to move the view
  • Click nodes to select them
  • Notice the colors: blue ‚Üí purple ‚Üí green = hierarchy
